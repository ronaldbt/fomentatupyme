# FomentaTuPyme

Sitio web de marketing digital para pymes del Biobío (Nuxt 3 + Tailwind).

## Desarrollo

```bash
cd frontend
npm install
npm run dev
```

## Producción

```bash
cd frontend
npm run build
```

El despliegue en Firebase Hosting usa `frontend/.output/public` (ver `frontend/firebase.json`).
